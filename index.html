Introduction

Welcome to SOS system. This Privacy Policy explains how the application collects, uses, stores and protects your personal information. The application is intended to help users send emergency SOS requests and receive assistance from nearby users.

The application requests access only to the permissions required for its functionality. No personal information is collected before the user grants the required permissions.

Information We Collect

Depending on how you use the application, we may collect the following information:

  • Telegram User ID used for authentication.
  • User profile information voluntarily entered by the user.
  • Current geographic location (only after permission is granted).
  • Firebase Cloud Messaging token used to deliver push notifications.
  • Technical information such as Android version, device model and application version.
  • Temporary authorization tokens required for secure login.

How We Use Information

Collected information is used exclusively for operating the SOS system.

  • User authentication.
  • Displaying nearby emergency requests.
  • Showing the user's own SOS request.
  • Sharing the user's location only when required by the emergency functionality.
  • Sending push notifications.
  • Preventing abuse and unauthorized access.
  • Improving application reliability and stability.

Legal Basis

Personal information is processed only when one of the following legal bases applies:

  • User consent.
  • Performance of the requested service.
  • Compliance with applicable laws.
  • Protection of the vital interests of the user during emergency situations.

User Consent

Before collecting personal information, SOS system requests the permissions required by Android. By granting permissions and continuing to use the application, the user expressly agrees to the collection, processing and storage of personal information described in this Privacy Policy.

The user may refuse to grant any permission. However, some functions of the application, including displaying nearby SOS signals or sending emergency requests, may not operate correctly without the required permissions.

Withdrawal of Consent

The user may withdraw consent at any time.

  • Disable permissions in Android Settings.
  • Delete the application.
  • Stop using the service.
  • Contact the developer requesting deletion of personal data.

After consent has been withdrawn, some or all application functions may become unavailable.

Sensitive Personal Information

The application may process location data because it is necessary for providing emergency assistance. Location information is used only for SOS functionality and is never sold to advertisers or used for personalized marketing.

Android Permissions

Permission Purpose
ACCESS_FINE_LOCATION Display nearby SOS requests and determine user location.
ACCESS_COARSE_LOCATION Approximate location when precise location is unavailable.
FOREGROUND_SERVICE_LOCATION Background location updates while emergency services are active.
POST_NOTIFICATIONS Receive emergency push notifications.
INTERNET Communication with the backend server.

Compliance with Huawei AppGallery

  • No personalized advertising is used.
  • No marketing profiling is performed.
  • No personal information is sold to third parties.
  • User consent is obtained before processing personal data.
  • The user may withdraw consent at any time.
  • Only permissions required for application functionality are requested.

Data We Do Not Collect

To protect user privacy, SOS system intentionally does not collect or access the following information:

  • Personal photos.
  • Personal videos.
  • SMS messages.
  • Call history.
  • Phone contacts.
  • Microphone recordings.
  • Calendar information.
  • Biometric information.
  • Payment information.
  • Personalized advertising identifiers.
SOS system does not sell personal information, does not use behavioral advertising and does not create marketing profiles.

Data Storage

The application stores only information required for providing the service.

  • Temporary login tokens are deleted after successful authentication.
  • Expired authorization tokens are automatically removed.
  • The application does not permanently store location history.
  • SOS location is stored only while required for emergency functionality.
  • Server logs are retained only as long as necessary to ensure security and reliability.

Data Retention

Personal information is retained only for the period necessary to provide the requested services, comply with legal obligations, resolve disputes and protect legitimate interests.

When information is no longer required, it is securely deleted or anonymized.

Data Security

SOS system applies reasonable technical and organizational measures to protect personal information against unauthorized access, loss, misuse, alteration or disclosure.

  • Secure authentication.
  • Encrypted HTTPS communication.
  • Restricted server access.
  • Access tokens for authorization.
  • Regular software updates.

Children's Privacy

The application is not specifically directed to children under the age required by applicable law. If a parent or guardian believes that a child has provided personal information, they may contact the developer to request deletion.

International Data Transfer

Depending on the user's location, information may be processed on servers located in different countries. Reasonable safeguards are applied to protect personal information during international transfers.

Политика конфиденциальности

Добро пожаловать в SOS system. Настоящая Политика конфиденциальности объясняет, какие данные собирает приложение, для каких целей они используются, как они защищаются и какие права имеет пользователь.

Приложение запрашивает только те разрешения, которые необходимы для его работы. Никакие персональные данные не собираются до получения согласия пользователя.

Какие данные мы собираем

  • Telegram ID для авторизации.
  • Информацию профиля, введённую пользователем.
  • Геолокацию (только после предоставления разрешения).
  • Firebase Cloud Messaging Token для push-уведомлений.
  • Техническую информацию об устройстве (версия Android, модель устройства, версия приложения).
  • Временные токены авторизации.

Для чего используются данные

  • Авторизация пользователя.
  • Работа системы экстренных SOS-сигналов.
  • Отображение пользователей поблизости.
  • Передача геолокации во время активного SOS.
  • Отправка push-уведомлений.
  • Защита от несанкционированного доступа.
  • Повышение безопасности и стабильности работы приложения.

Согласие пользователя

Перед обработкой персональных данных приложение запрашивает необходимые разрешения Android. Продолжая использование приложения и предоставляя необходимые разрешения, пользователь выражает согласие на обработку персональных данных, описанную в настоящей Политике.

Отзыв согласия

Пользователь может в любой момент:

  • отключить разрешения в настройках Android;
  • удалить приложение;
  • прекратить использование сервиса;
  • обратиться к разработчику с запросом на удаление данных.

После отзыва согласия часть функций приложения может стать недоступной.

  • Фотографии.

  • Контакты телефона.
  • Историю звонков.
  • Записи микрофона.
  • Платёжную информацию.
  • Данные для персонализированной рекламы.
SOS system не продаёт персональные данные, не использует персонализированную рекламу и не создаёт маркетинговые профили пользователей.

Используемые сервисы

  • Telegram — авторизация пользователей.
  • Firebase Cloud Messaging — push-уведомления.
  • OpenStreetMap — отображение карты.
  • Android Location Services — определение местоположения.

Хранение и защита данных

Персональные данные хранятся только в течение срока, необходимого для работы приложения. Используются защищённые соединения HTTPS, JWT-токены авторизации и современные методы защиты сервера.

隐私政策

欢迎使用 SOS system。 本隐私政策说明本应用如何收集、使用、存储和保护您的个人信息， 以及您所享有的相关权利。

SOS system 仅会申请实现应用功能所必需的权限。 在您授权之前，本应用不会收集您的个人信息。

我们收集的信息

  • 用于登录验证的 Telegram 用户 ID。
  • 用户主动填写的个人资料。
  • 位置信息（仅在您授权后获取）。
  • Firebase Cloud Messaging 推送通知令牌。
  • 设备基本信息，例如 Android 版本、设备型号和应用版本。
  • 用于安全登录的临时身份验证令牌。

信息使用目的

  • 完成用户身份验证。
  • 提供 SOS 紧急求助功能。
  • 显示附近的求助信息。
  • 在用户主动发起 SOS 时共享位置信息。
  • 发送推送通知。
  • 保障账户及系统安全。
  • 提高应用稳定性和服务质量。

用户同意

在收集任何个人信息之前， SOS system 会请求 Android 系统权限。 当您授予相关权限并继续使用本应用时， 即表示您同意按照本隐私政策 收集、使用、处理和存储您的个人信息。

撤回同意

您可以随时撤回授权：

  • 在 Android 设置中关闭相关权限；
  • 卸载本应用；
  • 停止使用本服务；
  • 联系开发者申请删除您的个人信息。

撤回授权后， 部分或全部功能可能无法继续使用。

Android 权限说明

权限 用途
